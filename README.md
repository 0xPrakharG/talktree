# TalkTree

## Getting Started

First, run the development server:

```bash
npm run dev
# or
bun run dev
```

## Key Features

```
📡 Real time communication
👍 Reactions to messages
🧵 Threads / Replies system
✏️ Editing messages
🗑️ Deleting messages
🔐 Role based access control
🖼️ Image attachments
🔒 Authentication with Next auth v5
📺 Channel creation
🏢 Workspace creation
✉️ Invite system / Invite codes
💬 Direct messaging
👥 User profiles
```
